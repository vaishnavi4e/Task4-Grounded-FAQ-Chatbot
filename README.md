# Grounded FAQ Chatbot using Gemini API

## 📌 Overview

This project is a Grounded FAQ Chatbot developed using Python, Google Colab, and Google's Gemini API.

The chatbot answers customer questions using a predefined knowledge base for **SkyWings Travel Agency** and refuses questions outside the knowledge base.

---

## ✨ Features

- 20 Frequently Asked Questions
- Multi-turn conversation
- Grounded responses using a knowledge base
- Refuses unrelated questions
- Uses Google Colab Secrets for API key
- Temperature set to **0.2**
- Continuous chat using `input()`
- Exit using `quit` or `exit`

---

## 🛠 Technologies Used

- Python
- Google Colab
- Google Gemini API
